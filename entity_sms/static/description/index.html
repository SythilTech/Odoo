Description

Allows 2 way sms conversations between leads/partners using the twilio gateway
Also supports other gateways if you can be bothered coding them...
Automaticlly creates new leads if a lead or partner doesn't have that mobile number
Built modularly so you can easily add a new gateway of your choice
Twilio gateway accounts require that you set allowed delivary countries

Individual gateway features
SMS GatewaySend SMSDelivary Receipts*Receive SMS*
twilioOOO
smsglobalOXX
smsgateway.caOXX

Note*
Real time receiving smses and delivary receipts require a live server, a scheduled action will also check for new smses but that’s only every hour.

Configure Multiple Accounts

Manage multiple gateways in the central location.

Instructions
1. Create a twilio/smsglobal/smsgateway account and add credit
2. Go to Settings->SMS Gateway Configuration->Accounts and enter the all the api settings

SMS Leads and Partners

Fast and straightforward smsing of leads and partners

Instructions
1. Setup an account and add credit
2. Enter mobile number in the lead/partner it will automaticlly get converted to e164 standard using the country
3. Simply select 'Send SMS' from the more menu and type in your message

SMS Templates

Use SMS templates to reduce the amount of repeated message.

Instructions
1. Go to Settings->SMS Gateway Configuration->SMS Templates
2. Select the target model, sms account and type away

SMS Auto responders

Automaticly send out response smses when a keyword sms is sent.
e.g. INFO=Thank you for requesting more information on our product

Instructions
1. Go to Settings->SMS Gateway Configuration->Auto Responders
2. Enter the target mobile number, Keyword and response sms

SMS History

View all the smses sent/received from the system.

Instructions
1. Go to Settings->SMS Gateway Configuration->SMS History for all smses
2. Messaging->SMS History for smses sent by you

Mass SMS

SMS a large amount of people at once.
Instructions
1. Setup an account and add credit
2. Go to Settings->Verified Numbers select/create a from number and hit "Verify Number" button
3. Go to Marketing->Mass SMS, select your records and mass send out
