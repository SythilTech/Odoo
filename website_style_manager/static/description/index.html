Description

Add site wide CSS styles and see the change in real time
Also let's you override LESS variables, so you can make sweeping changes to a theme

Apply site wide CSS styles

Instructions

  1. Go to Website->Customize->Custom CSS
  2. Add CSS rules to the text box and preview the changes in real time
  3. Hit save when you like the changes you have made
