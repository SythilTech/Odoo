Description

Make video calls with other users within your system
*IMPORTANT* Google Chrome requires the use of https to access camera and screensharing
*IMPORTANT* Google Chrome also needs to be started with the "--enable-usermedia-screen-capturing" command line flag to enable screensharing
*IMPORTANT* Mozilla Firefox requires https to access screensharing

Instructions

1. Click on the phone icon in the top right hand corner, it will display all users that have logged in atleast once
2. You can select between making a video call, audio only call or two way firefox screenshare
3. The other user will receive a notification to answer the call
4. Both users have to accept access to camera/audio
5. Once both users have accepted media access the call will begin
