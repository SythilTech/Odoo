Description

Adds the Twilio gateway to the sms framework

*NOTE* Real time receiving of smses requires that the server/database is accessible from the internet

Send and receive smses using Twilio

Send and receive smses using Twilio.

Instructions
1. Create a Twilio account here
2. Enter your account SID and auth token under Settings->Technicial->SMS->Accounts
3. Hit "Quick Setup" and your Twilio numbers will be entered into the system and inbound smses will go to your server
